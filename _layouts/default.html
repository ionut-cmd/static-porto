Secure Onboarding and Offboarding of IoT Devices

Software Developer - NquiringMinds

Contributed on implementing a secure onboarding mechanism for IoT devices, utilising a novel certificate-based authentication instead of traditional PSK methods. My contribution involved modifying a FreeRADIUS server to manage device onboarding and offboarding by revoking access for malicious devices. The project was executed on Raspberry Pi devices, where I designed and built a custom C server and a lightweight companion application to demonstrate the full onboarding and offboarding workflow. The solution aimed to enhance the security and efficiency of IoT device connectivity in real-world deployments.

CLI Tool for SBOM Generation and Vulnerability Detection in Multi-Language Projects

Software Developer - NquiringMinds

Developed a command-line interface (CLI) tool that automates the generation of Software Bill of Materials (SBOMs) and performs comprehensive vulnerability detection across various project types. This tool supports a wide range of project environments, including binary firmware, Docker images, and applications written in C/C++, and more. Designed to enhance security and transparency, the tool identifies potential vulnerabilities, enabling proactive security management for diverse software ecosystems. As a continuation of this project, I co-authored a publication, presenting the tool's framework and impact on modern software security practices..

Automated Website Generator for Spreadsheet-Based Content

Software Developer - NquiringMinds

Developed a website generator tool that converts spreadsheet documents into fully functional websites using YAML schemas and Docusaurus. This tool automates the transformation of structured data into web content, significantly streamlining content management and publication processes for organizations. The tool was primarily built in JavaScript and enables non-technical users to generate websites directly from spreadsheets, with data validation and formatting handled via YAML. This solution is currently used by the Internet of Things Security Foundation (IoTSF), a prominent UK organization dedicated to advancing IoT security standards. The project highlights my skills in JavaScript, YAML, and Docusaurus, as well as my ability to create scalable, user-friendly tools.
